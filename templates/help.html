How to Use MarkovMe

What Data Is Used

Facebook: personal timeline posts.

Plaintext: user provided text.

Twitter: one/two public twitter handles.

Usage

Facebook: No data is stored by the app externally, and it is only used while you are on the page. Multiple posts can be quickly generated by pressing the button each time a new one is desired.

Plaintext: The data entered by the user is added to the database to generate a sentence based on what was entered. Enter sufficient data for the post to be successfully generated.

Twitter: 200 tweets are taken per person to generate a new tweet persona.
